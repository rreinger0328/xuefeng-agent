# 雪峰Agent 提问模板

> 复制模板，把【】里的字换成你自己的情况，直接发给 Agent。

---

## 高考生选专业（最常用）

```
我是【省份】考生，选科【物理/历史】，
今年考了【分数】分，全省排名大概【位次】名。
家里是【普通工薪/做生意的/爸妈在XX行业】，
想去【城市或省份】，最讨厌学【数学/背书/都还行】，
毕业想【找个好工作/考公务员/考研/稳定就行】。
帮我推荐专业和学校。
```

---

## 不知道学什么（先理方向）

```
我是【省份】的，今年高考，【物理/历史】类考了【分数】分。
完全不知道学什么好，也不确定想去哪。
你先问我几个问题，帮我理一下方向。
```

---

## 有目标了，让 Agent 评估

```
我想学【专业】，想去【学校或城市】。
我是【省份】【物理/历史】类，考了【分数】分，排【位次】名。
你帮我看看这个方向合不合适？有什么坑？
```

---

## 几个选项纠结中

```
我在纠结：
1. 【学校A】的【专业A】
2. 【学校B】的【专业B】
我是【省份】【物理/历史】类，【分数】分，排【位次】。
帮我对比一下哪个更适合我？
```

---

## 专科/低分段

```
我只考了【分数】分，只能上专科了。家在【省份】。
就是想毕业能有个稳定工作。帮我看看学什么好？
```

---

## 高一选科

```
我现在高一，马上选科。以后想往【医学/计算机】方向走。
各科大概【简单说下强弱】。选科怎么选？
```

---

## 小技巧

1. **位次比分数重要** — 有全省排名一定带上
2. **讨厌什么和喜欢什么一样重要** — 说清楚不能接受的，Agent 帮你避开
3. **家里干什么的别不好意思说** — 爸妈在电力、铁路、医院、学校、做生意的，这决定了很多隐藏的好路
4. **多聊几轮效果更好** — Agent 追问的时候认真答，聊得越深建议越准
