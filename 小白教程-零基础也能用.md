# 雪峰Agent 使用教程（小白版）

---

## 第一步：装 Python

1. 打开浏览器，输入 **python.org**
2. 点黄色大按钮下载
3. 双击打开，⚠️ **勾上底部的 Add Python to PATH**，不勾后面用不了
4. 点 Install Now，等两分钟装完

---

## 第二步：下载雪峰Agent

1. 打开 **github.com/ziqihe10-droid/xuefeng-agent**
2. 点右侧绿色 **Code** 按钮 → **Download ZIP**
3. 下载完右键解压到桌面
4. 打开解压出来的文件夹

---

## 第三步：搞个 DeepSeek Key

1. 打开 **platform.deepseek.com**，注册登录
2. 左侧点 **API Keys** → 创建 → 复制 `sk-` 开头的那串（只显示一次，保存好）

---

## 第四步：启动

1. 双击文件夹里的 **启动.bat**
2. 浏览器会自动打开，没打开就手动输入 `http://127.0.0.1:8765/`
3. 点右上角红色 **API设置**：
   - Base URL：`https://api.deepseek.com`
   - API Key：粘贴你的 `sk-` 钥匙
   - Model：`deepseek-chat`
   - Tavily Key：选填，推荐去 **tavily.com** 免费注册一个
4. 点 **保存并测试**，显示绿色连接OK就行

## 常见问题

**双击启动闪退？**
→ 没装 Python 或没勾 Add to PATH。重装，必须勾那个框。

**浏览器打开了但是空白？**
→ 等几秒刷新。或者地址栏输 `http://127.0.0.1:8765/`

**按钮点了没反应？**
→ 按 `Ctrl+Shift+R` 强制刷新

**以后怎么用？**
→ 双击 `启动.bat`，浏览器自动打开，开问
