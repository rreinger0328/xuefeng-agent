# AI 高考志愿顾问 知识库

> 基于多年高考志愿辅导经验、院校数据、就业报告等公开信息综合整理
> 数据以当年官方发布为准，建议结合最新招生简章使用

---

## 一、核心咨询哲学

### 1.1 三大基石

| 原则 | 内容 |
|------|------|
| **选择 > 努力** | 方向选错，再努力也是白费。高考志愿和考研选择，比考试本身更影响人生走向 |
| **就业倒推法** | 不从"我喜欢什么"出发，而从"什么能让我找到好工作"倒推 |
| **阶层现实主义** | 承认家庭资源差异，给出符合家庭实际的建议，不给"鸡汤" |

### 1.2 灵魂追问法（咨询时必问）

1. **你家是哪的？**（省份 → 决定竞争池）
2. **你考了多少分/多少位？**（硬数据，一切推荐的基础）
3. **你想学什么？**（兴趣方向，但可被现实修正）
4. **你想去哪？**（地域偏好 + 限制）
5. **你家里干什么的？**（家庭资源，决定某些行业能不能进）
6. **你以后想干什么？**（就业、考公、深造、稳定？）

### 1.3 家庭背景判断矩阵

| 家庭背景 | 推荐策略 |
|---------|---------|
| 有行业资源（电力/铁路/医疗/教育） | 优先继承家庭资源，选对口专业 |
| 普通家庭无资源 | 技术类专业优先（计算机/电气/机械/医学） |
| 富裕家庭 | 可考虑商科/管理/艺术，容错率高 |
| 农村/贫困家庭 | 优先选能"翻盘"的专业（医学/计算机/师范） |

## 二、高考志愿填报方法论

### 2.1 信息采集五步（不可跳过）

```
第一步：硬分 — 省份 + 分数 + 位次 + 选科
第二步：兴趣 — 想学什么大类？最讨厌什么？
第三步：地域 — 想去哪？不去哪？有没有特殊限制？
第四步：家庭 — 父母做什么的？有什么资源？
第五步：诉求 — 就业第一？考公第一？深造第一？稳定第一？
```

每步信息不全，不给出结论。

### 2.2 冲-稳-保 梯度规则

#### 新高考模式一（一校一专业，浙江/山东/河北/重庆/辽宁）

```
冲：上浮 15000 位次以内（例：位次30000，冲15000-30000位内院校）
    高概率冲击：上浮 10000 位次
稳：下浮原位次 + 0~3000 位（第一阶梯）
              + 3000~6000 位（第二阶梯）
              + 6000~9000 位（第三阶梯）
保：第一阶梯：原位次 +9000~15000 位
    第二阶梯：原位次 +15000~30000 位
```

#### 新高考模式二（专业组填报，江苏/北京/上海/湖北/湖南/广东/福建/天津/海南）

```
冲：上浮 15 分以内（用转换分，参考近1年分数）
稳：下浮 0~5 分（第一阶梯），5~10 分（第二阶梯）
保：下浮 15~30 分，至少 2-3 个
```

#### 老高考模式（各省具体规则）

```
核心不变：冲-稳-保
- 冲击院校如有不满意专业，切忌冲击
- 不是所有人都需要冲击院校，如无满意冲击院校，以保专业为主
- 保底学校至少 1-2 个
```

### 2.3 位次法（核心技巧）

高考出分后 → 查一分一段表 → 得到位次 → 参考前1-2年同位的录取情况

```
位次比分数更稳定
- 分数每年波动
- 位次是你在全省的相对位置，更可靠
- 重点参考前 1-2 年专业分数位次
- 3年位次变化看趋势，不只盯单年
```

### 2.4 服从调剂原则

```
冲击院校 → 谨慎服从（可能被调剂到冷门专业）
稳妥院校 → 酌情服从
保底院校 → 必须服从（先保住学上）
```

**关键原则**：冲击院校若有不满意的专业，切忌冲击。不是所有学生都必须冲击。

## 三、专业选择知识库

### 3.1 12大学科门类核心摘要

#### 01 哲学类
- **就业难度：极高**。最典型"就业困难"专业，无直接对口岗位
- 方向：考公（党校/宣传部）、教育（需要深造）、新闻出版（独特视角写作）
- **观点**：普通家庭慎选，除非走学术/考公方向明确

#### 02 经济学类（热门）
- 含经济学、金融学、金融工程、国际经济与贸易
- 金融学/金融工程 → 银行/证券/保险/基金（金融工程需数学好）
- 国贸 → 涉外贸易/外企/跟单/报关
- **观点**：金融看家庭资源，银行系统"关系户"多。普通家庭学金融工程（偏量化技术）比纯金融好

#### 03 法学类
- 含法学、政治学、社会学、民族学、马克思主义理论、公安学
- 法学 → 公考/律所/企业法务（都需通过法考）
- 公安学 → 公安/检察/法院/司法
- **观点**：法学是"考公第一专业"，但法考通过率低。五院四系优先

#### 04 教育学类
- 教育学 ≠ 当老师！教育学是研究教育规律的社会科学
- 当老师要考的是学科教学类（如学科教育·数学）
- 学前教育/小学教育/特殊教育 → 学校/教育部门

#### 05 文学类
- 中文 → 考公/文秘/宣传/编辑/教师
- 外语 → 翻译/外企/外贸/教育
- 新闻传播 → 报社/电视台/新媒体/考公
- **观点**：中文和新闻是"考公大户"。外语建议选大语种，小语种就业受限

#### 06 历史学类
- **就业难度：高**。对口岗位少，待遇偏低
- 方向：教学/科研/考古/博物馆/考公

#### 07 理学类（基础学科）
- 数学/物理/化学/生物/地理/统计/心理
- 关键出路：
  1. 科研（需深造读研读博）
  2. 教师（考教师资格证）
  3. 跨专业考研（理工科基础好，转到工科/经管受欢迎）
- **观点**：理学是"万金油底座"，适合深造或做研究的人

#### 08 工学类（最大门类，最好就业）
- 几乎所有工科都有对口就业行业
- 热门：计算机、软件工程、电子信息、电气工程、机械工程、土木工程
- **观点**：工科是普通家庭孩子最稳妥的选择。"学一门手艺，走遍天下都不怕"

#### 09 农学类
- 植物生产/动物医学/林学/水产/草学
- 对口：农业/园林/畜牧/林业/环保部门
- **观点**：分数不高的孩子可以关注，但不是"来钱快"的专业

#### 10 医学类（对口率最高，转行最难）
- 临床/口腔/基础/预防/中医/药学/护理
- **核心特点**：在哪儿上学基本就在哪儿就业，地域性极强
- 护理学 → 各级医院护士
- **观点**：医学是"稳定但辛苦"的选择，不能只看收入

#### 11 管理学类
- 会计学/财务管理/审计/人力资源 → 相对好就业
- 信息管理/档案学/农林经济管理 → 就业较难
- **观点**：管理学"不只是管人"，扎实学一门技能（会计/审计）比学纯管理有用

#### 12 艺术学类
- 音乐/舞蹈/戏剧/美术/设计
- **观点**：烧钱，家里有实力再考虑。设计类（UI/视觉传达）相对好就业

### 3.3 如何判断一个专业好不好（方法论）

1. **看专业的开设时间** — 历史悠久 vs 新设专业
2. **看课程设置** — 有无全英语课程/专业选修课数量和种类
3. **看培养目标** — 宽口径人才 vs 技能型人才
4. **看专业在学校的地位** — 重点学科/一流学科/特色专业
5. **看就业方向** — 因研究方向不同，就业领域差异巨大
6. **看博士点/硕士点** — 有硕士点博士点说明该专业实力强
7. **看院士数量** — 有院士坐镇的学科，实力不会差

## 四、学校选择方法论

### 4.1 如何看待学校排名

- 大学排行榜有参考价值但不是唯一依据
- 要结合专业排名、地域、行业认可度综合判断
- 985/211/双一流 是硬标签，在就业市场上仍然很有分量

### 4.2 如何了解一所大学好不好

1. **从学校历史看强势专业** — 合并前的"前身"决定了优势学科
2. **从院士看强势专业** — 院士所在学科就是该校王牌
3. **从博士点/硕士点看** — 有硕博点的专业说明实力雄厚
4. **名字有特点的学校** — 如"XX邮电""XX电力""XX交通"，行业属性明显
5. **"窝里横"大学** — 在省内认可度很高，出省就不一定
6. **部级直属院校** — 行业资源丰富，就业渠道畅通

### 4.3 学校-专业-城市 三角关系

```
高分学生 → 学校优先（985/211 标签带来的机会多）
中等学生 → 专业优先（技术类专业保就业）
分数偏低 → 城市优先（大城市实习机会多，看涨）
```

**经典问题**：好专业和好学校应该选哪个？
- 工科类：专业排名 > 学校排名
- 社会人文学科：学校排名 > 专业排名
- 医学/农学：地域 > 专业排名 > 学校排名

## 五、城市选择逻辑

### 5.1 核心原则

1. **在哪上学，大概率在哪就业**（医学尤甚）
2. **大城市提供更多实习和就业机会**
3. **产业集群匹配** — 学芯片去上海/合肥，互联网去深圳/杭州/北京
4. **离家距离** — 因人而异，有的家庭希望近，有的无所谓

### 5.2 城市产业集群速查

| 产业方向 | 推荐城市 |
|---------|---------|
| 互联网/科技 | 北京、深圳、杭州、成都 |
| 芯片/半导体 | 上海、合肥、无锡 |
| 金融 | 上海、北京、深圳 |
| 汽车 | 长春、上海、广州、武汉、重庆 |
| 航空航天 | 北京、西安、哈尔滨、成都 |
| 医药 | 上海、苏州、泰州、连云港 |
| 铁路/交通 | 北京、成都、长沙、大连 |

## 六、考研方法论精华

### 6.1 考研选校选专业核心逻辑

```
① 先定专业 → ② 再选地区 → ③ 再定学校档次 → ④ 看考试科目 → ⑤ 看招生人数 → ⑥ 看隐性信息
```

### 6.2 专业选择

- **先定专业，后定学校**（与高考不同，考研先报名后考试）
- 跨专业要慎重，了解目标专业的考试科目门槛
- 学硕 vs 专硕选择：看你是想搞科研还是想就业

### 6.3 学校选择

- 专业定了后，筛选招生单位（全国800+研招单位）
- 优先考虑自己熟悉的地区
- 查看目标院校研究生院网站 → 招生专栏
- 关键信息：招生简章、招生专业目录、报录比、推免比例
- **隐性信息**：目标院校目标专业的学院网站（可能包含导师信息/研究方向）

### 6.4 信息检索三件套

1. **中国研究生招生信息网（研招网）**：报名 + 调剂 + 查学校
2. **目标院校研究生院网站**：招生简章 + 招生专业目录 + 推免信息
3. **目标专业所在学院的学院网站**：导师信息 + 研究方向 + 往年复试名单

### 6.5 考研常见问题

| 问题 | 答案要点 |
|------|---------|
| 数学不好考什么专业 | 哲学/政治学/历史/教育/文学/管理类部分（不考数学但竞争激烈） |
| 理工科找工作还是考研 | 想做技术研发 → 必须考研；做一线生产 → 可先工作 |
| 电子信息/计算机考研难 | 竞争激烈是事实，考不上好的可以选"地域好+专业有特色"的学校 |
| 三本能考211吗 | 可以，但要在初试中绝对优势 + 提升综合能力 |
| 要不要二战 | 取决于考研动机：良心发现 vs 形势所迫；形势所迫就别二战 |
| 城市怎么选 | 医学/农学 → 未来就业地；理工 → 产业集群地；社科 → 大城市 |
| 学校排名 vs 专业排名 | 医学/农学：地域最重要；理工：专业排名重要；社科：学校排名重要 |

## 七、咨询表达风格要素

### 7.1 语言特征

- **反问句多**："你家干什么的？你凭什么觉得你能进？"
- **大白话**：不用学术词汇，用家长都能听懂的比喻
- **东北味儿**：适当加入"那啥""整"等东北口语
- **幽默扎心**："你这个分数还想学金融？你家里有矿还是有银行？"
- **数据+段子**：先甩数据，再甩段子，最后给建议

### 7.2 经典语录模式

```
"你以为XX专业是学YY的，实际是学ZZ的，出来只能干WW"
"别一上来就问哪个学校好，你先告诉我你是谁，你家是谁"
"选择比努力重要，不是说你努力了就一定能考上"
"XX专业，不是不能报，是你这种家庭条件不建议报"
```

### 7.3 表达禁忌（我们的 agent 要避免的）

- ❌ "综合评估""维度分析""量化考量" — 太学术
- ❌ 过于啰嗦的长篇大论 — 简洁有力才是王道
- ❌ 一味说好话不给风险提示 — 要敢说"这专业不行""这学校别去"
- ❌ 不追问直接给结论 — 这是咨询最核心的风格

## 八、数据可信度分级（输出时标注）

| 级别 | 含义 | 示例 |
|------|------|------|
| **T1** | 官方公开数据 | 各省教育考试院一分一段表、录取批次线 |
| **T2** | 高校官方数据 | 各校招生网公布的招生计划、录取分数线 |
| **T3** | 经验规律/方法论 | 冲稳保规则、位次法、专业选择逻辑 |
| **T4** | LLM 训练数据/推测 | 非精确的就业趋势、行业判断 |

**输出原则**：T1-T2 级数据给出具体数字，T3-T4 级数据加"以上基于往年趋势，具体请查当年官方公布"的声明。

## 九、咨询服务的 output 格式

### 最终推荐应包含

```
📊 你的画像：{省份} {选科} {分数/位次}，家庭{资源情况}，想去{地域}，诉求{就业/深造/考公}
```

## 十三、稳定就业路径详解

> 教师、医生、公务员、央国企是家长最关心的"稳定就业"方向。每条路径的进入门槛、薪资待遇、适合人群各不相同。

### 13.1 教师路径

**编制类型**：全额拨款事业编、差额拨款事业编、备案制（员额制）、合同制/派遣制。稳定性依次递减。

**成为教师的途径**：
| 途径 | 说明 |
|------|------|
| 公费师范生 | 国家/省级公费，免学费住宿费+补贴，毕业包分配有编 |
| 优师计划 | 面向中西部欠发达地区，定向培养 |
| 普通师范生考编 | 自行参加教师招聘考试（笔试+面试） |
| 特岗教师 | 服务农村学校3年，期满可转编 |

**教师收入参考**：
- 基本工资 + 绩效 + 年终奖
- 五险一金缴存比例较高
- 寒暑假带薪（年休假超百天）
- 2021年全国教师总数1792.97万人，年增长3.52%

**适合人群**：追求稳定、规律作息、喜欢与人打交道。对女性来说社会认可度高。

### 13.2 医生路径

**提前批定向医学生**：入学前就预定工作去向，免学费+包分配，但必须服务基层指定年限。

**成为医生的门槛**：
- 必须通过执业医师资格考试（否则属非法行医）
- 医学本科5年 + 规培3年 = 至少8年
- 发达城市三甲医院基本要求博士学历

**学医劝退理由**：
- 学制长（5年本科+3年规培+3年专硕=11年起步）
- 35岁以前基本不赚钱
- 工作强度大、夜班多
- 医患关系复杂

**多元出路**：除临床医生外，还可选择医药企业、医疗器械公司、医疗保险、医学编辑、医学教育等。

### 13.3 公务员路径

**国考 vs 省考**：

| 维度 | 国考 | 省考 |
|------|------|------|
| 招录单位 | 中央部委、直属机构 | 省/市/县级机关 |
| 考试时间 | 每年11-12月 | 各省不一 |
| 竞争比 | 2024年约57:1 | 因省而异 |
| 岗位特点 | 平台高，发展空间大 | 离家近，基层多 |

**选调生**：竞争更少但要求更高。仅限应届生+党员/学生干部，且对学校层次有要求（很多要求双一流）。

**大学四年考公规划**：
- 大一：了解考公基本要求，保持学习成绩
- 大二：入党、积累学生干部经历
- 大三：开始看行测和申论，刷真题
- 大四：参加国考（10月报名）和省考

**适合考公的专业**（普适型）：
- 法学类（公检法+所有单位都需要）
- 汉语言文学（文字岗位多）
- 财会类（审计/财政/税务）
- 计算机类（各单位信息化需求大）

### 13.4 央国企系列

#### 国家电网/南方电网
- **对口专业**：电气工程及其自动化（核心）、能源与动力工程
- **院校优势**：华北电力大学、原电力部直属院校（东北电力、上海电力、三峡大学等）
- **特点**：稳定性高、待遇好、但大城市竞争激烈。县级以下更容易进。
- **招聘**：校招为主，一本以上电气专业基本稳

#### 石油系统（三桶油：中石油/中石化/中海油）
- **对口专业**：石油工程、地质工程、化工、机械
- **院校**：中国石油大学（华东/北京）、西南石油大学、东北石油大学
- **特点**：待遇好但地点偏，常驻油田

#### 铁路系统
- **对口专业**：交通运输、土木工程（铁道方向）、机械工程、电气工程
- **院校**：西南交通大学、北京交通大学、石家庄铁道大学、大连交通大学、兰州交通大学
- **岗位**：车务（车站运营）、机务（机车驾驶/维护）、工务（线路维护）、电务（信号/通信）、车辆（车厢维护）
- **特点**：稳定性高、福利好、但工作地点可能偏远

#### 中国烟草
- **特点**：待遇优厚（传说中的"金饭碗"），但竞争极其激烈
- **招聘**：烟草系统不看专业不看学校，招聘透明度相对较低
- **对口院校**：郑州轻工业大学（烟草学院）、云南农业大学

#### 航空航天/军工系统
- **对口专业**：航空航天类、兵器类、核工程类、电子信息类
- **院校**：国防七子（哈工大、北航、北理工、西工大、南航、南理工、哈工程）
- **特点**：有情怀、稳定、待遇中上、但部分涉及保密限制

#### 银行系统
- **机构**：央行（人民银行）、政策性银行（国开行/进出口银行/农发行）、国有商业银行（工农中建交）、股份制银行
- **对口专业**：金融学、经济学、会计学、计算机（金融科技方向）
- **招聘**：校招为主，笔试+面试，对学校层次有一定要求
- **注意**：银行基层工作压力不小，有业绩考核

### 13.5 稳定就业通用原则

1. **编制≠铁饭碗**：并不是进入央企国企都有编制，很多是合同制/同工不同编
2. **简历关**：央企国企也要筛选简历，不符合条件连考试资格都没有
3. **真正的铁饭碗**：不是在一个地方吃一辈子饭，而是一辈子到哪儿都有饭吃
4. **信息渠道**：国家公务员局官网、各国企官网招聘专栏、高校就业信息网

## 十四、专科志愿填报策略

> 专科≠没前途。专科志愿匹配好专业+学校+省份，就业不比普通本科生差。

### 14.1 专科选专业的四大原则

1. **就业导向优先**：专科以就业为首要目标，选实操性强、行业需求大的专业
2. **避开"虚"专业**：市场营销、行政管理等无技术壁垒的专业，本科都不好就业，专科更难
3. **选有行业壁垒的技能**：电力、铁路、医护、汽修、数控、烹饪等有证书或技能门槛的专业
4. **关注政策红利**：职业教育迎来政策红利期（十四五规划明确发展职业教育）

### 14.2 专科选学校的标准

- **双高计划院校**（197所）：专科中的"双一流"，师资和就业资源最好
- **国家示范性高职院校**（100所）：老牌强校
- **行业特色院校**：电力特色（各省电力高职）、铁路特色（铁道职院）、公安院校等
- **升本率**：专升本是很多专科生的路径，学校升本率高意味着学习氛围好

### 14.3 专科选地域的策略

- **根据专升本政策选省份**：各省专升本政策差异大，有的省升本容易、有的省限制多
- **根据产业选城市**：制造业强的城市（苏州/东莞/佛山）对技术工人需求大
- **电力/铁路特色学校**：在本省行业内认可度高，就近就业优势明显

### 14.4 专科报考技巧

- **分数和位次都重要**，但专科段更看位次
- **平行志愿规则**与本科相同，冲-稳-保逻辑适用
- **三个批次可报**：提前批（定向军士/免费医学定向等）、普通批、征集志愿

### 14.5 常见误区和补救

| 误区 | 真相 |
|------|------|
| "专科随便报都一样" | 专科学校和专业差异巨大，好的专科专业就业远超普通本科 |
| "专科不能考公" | 专科可报部分公务员岗位（基层岗位、公安辅警等） |
| "专升本没用" | 专升本第一学历是本科，对考公/考研/就业都有实质性帮助 |

### 15.1 高一：适应+选科

## 十六、2025-2026年最新趋势与实时信息

> ⚠️ 本模块信息具有时效性，agent 回答用户问题时应优先搜索最新数据，以下内容仅作基准参考。

### 16.1 2026年高考政策重大变化

**新高考改革收官**：29个省份已实行新高考，2026年为改革收官之年。西藏和新疆最后一年采用传统文理分科模式。

**命题方向转变**：
- "反押题、反套路"——死记硬背和题海战术效果大幅减弱
- 从"解题"转向"解决问题"——考查独立思考与真实情境应用能力
- 融入脑机接口、具身智能等科技前沿素材
- 提升科学素养和人工智能素养
- 杜绝偏题、怪题、超标题，保持难度稳定

**招生录取变化**：
- 严控大类招生，原则上按专业招生（减少调剂风险）
- 扩大优质本科招生规模
- 招生向基础学科、新兴学科、交叉学科和国家战略急需领域倾斜
- 推进"阳光志愿"信息服务系统下沉基层

**报名人数**：2026年约1290万考生。

### 16.2 AI对专业选择和就业的颠覆性影响

**AI替代风险排行**（Anthropic 2026年报告）：
- 计算机程序员：约40%工作任务可被AI覆盖，自动化覆盖率74.5%
- 客服：70.1%
- 数据录入员：67.1%
- 翻译、基础摄影、基础会计、法律基础服务也在高危之列

**计算机专业就业质量急剧下降**：
| 年份 | 绿牌专业中计算机类数量 |
|------|------------------------|
| 2021年 | 4个 |
| 2022年 | 2个 |
| 2023年 | 1个 |
| 2024-2025年 | 0个 |
- 计算机类毕业生月收入出现负增长（6863元→6846元）
- 40人毕业生仅1人拿到大厂offer（三四年前是30人）
- 但顶尖AI岗位月薪3万-5万+，年薪可达百万——行业内部分化极度严重

**高校专业大洗牌**（2024年度）：
- 全国撤销本科专业点1428个、停招2220个、新增1839个，规模历史之最
- 多省发布过剩专业预警：计算机、软件工程、英语、国贸、法学、市场营销等屡被点名
- 新增方向：569所院校开办AI本科专业，具身智能、智能医学工程、AI教育等

**核心结论**：AI淘汰的不是某个专业，而是只会重复机械劳动、拒绝拥抱变化的人。就业核心竞争力转向：批判性思维、创造力、共情力、复杂决策、跨领域整合能力。

### 16.3 2025-2026年就业市场新格局

**两大就业赛道**：
1. **AI原生赛道**（造AI、用AI）：大模型训练师、提示词工程师、AI产品经理等
2. **"专业+AI"融合赛道**（垂直领域最懂AI的人）：智能医疗、智慧农业、AI+金融等

**传统热门专业重新评估**：

| 专业 | 当前状态 | 建议 |
|------|---------|------|
| 计算机科学与技术 | 就业质量下降但高端人才稀缺 | 不能只学基础，必须+AI/大数据方向 |
| 软件工程 | 初级码农过剩，架构师稀缺 | 名校+深造，普通院校慎选 |
| 电子信息/集成电路 | 国家战略重点，持续看好 | 芯片方向是最稳妥的硬科技 |
| 电气工程 | 电网+新能源双驱动，稳定 | 进电网仍是很好的选择 |
